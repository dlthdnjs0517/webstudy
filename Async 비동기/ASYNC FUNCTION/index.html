Async keyword

async의 정의

async:

함수를 비동기 함수로 선언하는 키워드 함수명 앞에 async을 입력해 함수를 비동기 함수로 선언하면 함수를 자동으로 promise로 변환함

sing()
		.then((data) => {
		console.log("promise fulfilled with :", data)
		})
		.catch(err => {
		console.log("OH NO, Promise rejected!")
		console.log(err)
		})

결과:

	OH NO, Promise rejected!
	OH NO, PROBLEM!
	에러가 발생했기 때문에 catch 함수를 콜백한 것을 확인 할 수 있음.
